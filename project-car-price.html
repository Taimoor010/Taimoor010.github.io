Car Price Prediction App

A live machine learning tool that estimates used car prices from marketplace listing data.

Try Live Tool View Projects GitHub
Live

Problem

Used car buyers and sellers often struggle to estimate fair market value because prices vary by model, year, mileage, engine size, city, registration city, colour, variant, transmission, and other listing signals.

Approach

I collected marketplace listing data, cleaned and filtered the dataset, engineered relevant features, compared machine learning models, evaluated prediction error using MAE and R2, and deployed the final model into a live interface where users can enter car details and receive a predicted price estimate.

Input fields supported

Make/modelYearMileageEngine sizeCityRegistration cityColourVariantTransmission

Impact

  • Built and deployed a live ML price estimation tool.
  • Created a reusable workflow for data collection, cleaning, model training, and prediction.
  • Converted raw marketplace listings into a practical decision-support product.
